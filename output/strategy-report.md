## 0. 경영진 요약

GLP-Care는 한국 GLP-1 자가기록 유저(약 20~30만 명, 2026년 추정)와 중장년 복약관리 유저라는 **두 개의 인접 니치**를 연결하는 포지션에 있다. 글로벌 GLP-1 앱(Shed·Found·WW)은 한국 미진출이고, 한국 복약 앱(약 수첩류)은 GLP-1 맥락이 없으며, 범용 칼로리 앱(MyFitnessPal·팻시크릿)은 약물 데이터가 없다. 이 3각 공백이 GLP-Care의 해자다.

그러나 현재 빌드는 **데이터 영속성, 푸시 알림, 접근성 기초**가 모두 미구현 상태로 네 명의 테스트 페르소나 모두 3개월 내 이탈을 예측했다. 출시 전 P0 5개 이슈(데이터·푸시·폰트·터치·어제 배너)만 해결하면 기본 신뢰선을 회복할 수 있고, 이후 "약 + 칼로리 상관 인사이트"와 "GLP-1 증량 이력"이라는 2개 차별화 무기로 경쟁 우위를 구축한다.

수익화는 **무료 백본 + 프리미엄 고가치 기능**의 분리형 모델을 권장한다. 무료는 복약 기록·알림·달력·주간 요약까지 포함해 Medisafe형 "2개 약 이상 paywall" 실수를 피하고, 프리미엄은 진료용 PDF·가족 공유·약↔칼로리 상관 대시보드·iCloud 백업으로 구성한다. 가격은 **월 3,900원 / 연 29,000원**(한국 건강앱 지불감 기준, 연 갱신 시 30% 할인)이며, GLP-1 사용자 전용 "파워팩" 7,900원 옵션으로 증량 관리·체중 연동을 추가한다.

출시 후 3개월 로드맵은 (1) 6주차 100명 PMF 확인, (2) 10주차 프리미엄 도입, (3) 12주차 구독 전환율 3% 목표다. 핵심 KPI는 D30 리텐션 30%, DAU/MAU 25%, 3개월 구독 전환 3~5%.

## 1. GLP-Care 강점 정의

### 1.1 경쟁사에 없거나 약한 것

페인포인트 매트릭스의 "경쟁사 공백" 분석에서 도출된 4개 기회 영역을 GLP-Care의 강점으로 재정의한다.

**강점 1 — 한국 로컬 × GLP-1 특화 "원앱"**
Shed, Found, WW, Embla 등 글로벌 GLP-1 앱은 모두 한국 미진출이며 한국어 지원도 없다. 한국 시장 내에서는 삼성 헬스·카카오 PASTA가 범용 헬스 앱이지만 GLP-1 증량 일정, 주사 패턴, 자가구매 맥락을 반영하지 못한다. 한국어 + GLP-1 특화 기록이라는 포지션 자체가 현재 공백이다.

**강점 2 — 약물 + 칼로리의 통합 기록 (단, 현재는 "한 앱에 있을 뿐" 수준)**
복약 앱(Medisafe, MyTherapy, CareClinic)은 칼로리 DB가 없고, 칼로리 앱(MyFitnessPal, 팻시크릿, 인아웃)은 약물 데이터가 없다. GLP-Care는 구조적으로 두 가지를 한 앱에서 처리할 수 있는 유일에 가까운 위치다. 다만 **현재 구현은 두 기능이 물리적으로만 같이 있을 뿐 연결 인사이트가 없다** — 이것이 가장 큰 "구현해야 할 차별화"이자 Phase 2의 최우선 과제다.

**강점 3 — "기록만 하는" 가벼움 = 규제 회피 + 빠른 온보딩**
Shed·Found·WW Clinic은 의료팀 / 처방 / 약물 배송까지 끼워 규제 비용이 크고 유저 진입장벽도 높다. GLP-Care는 "유저가 직접 등록한 데이터를 시각화만" 하므로 의료기기·건강기능식품 법규 회피, 첫 사용까지 30초, 월 상담료 없음이 장점이다. 이는 "편의성"이 아니라 전략이다. 경쟁사가 따라 하려면 기존 사업을 포기해야 한다.

**강점 4 — GLP-1 증량 스케줄의 구조적 이해**
위고비 0.25→0.5→1.0→1.7→2.4mg, 삭센다 0.6→1.2→1.8→2.4→3.0mg, 리벨서스 3→7→14mg. 이 프로토콜은 표준화되어 있지만 복약 앱은 단순 "약 이름 + 용량"만 지원해 증량 시점의 소급 변경이 일어난다. GLP-Care v1.1의 DosageHistory 모델은 이 문제를 직접 해결한다. 자가구매 유저(페르소나 C 미경)가 자기 기억에 의존하지 않아도 되는 유일한 앱이 될 수 있다.

### 1.2 현재 구현 중 경쟁 우위 있는 것

정직하게 말하면 **현재 빌드의 경쟁 우위는 아직 거의 없다**. 강점 1~4 모두 "포지션 기회"이지 "구현된 무기"가 아니다. 현재 빌드에서 다른 앱 대비 낫다고 말할 수 있는 것은:

- 한국어 네이티브 UX (Shed·Found·Medisafe 영어 기반 대비)
- 광고 없음 (삼성 헬스·PASTA·팻시크릿 대비 — 단, 유지 정책 확정 필요)
- 복약 + 칼로리가 물리적으로 한 앱 (물리적 통합만으로도 신규 유저가 "원앱 시도"할 명분)
- 기획서 v1.1의 MedicationSchedule·DosageHistory 설계 (구현 완료 시 진짜 무기)

이 목록은 짧고 약하다. 그래서 P0·P1을 빠르게 처리해 "참가비"를 내고, P2에서 무기를 장착해야 한다.

### 1.3 "이 앱을 쓰는 이유" — 핵심 가치 제안 3개

경쟁사 조사와 페르소나 테스트에서 반복적으로 확인한 3개 가치 제안. 마케팅 카피·앱스토어 설명·온보딩 카피에 동일 문구로 사용 권장.

**VP1 — "위고비·오젬픽·리벨서스, 한국어로 기록하는 유일한 앱"**
경쟁사 미진출 공백을 직접 공략. GLP-1 약물 이름을 전면에 걸어 검색 유입 확보. "한국어로"가 핵심 단어 — 영어 앱 기피 유저(정호·B) 타겟.

**VP2 — "약을 챙긴 날이 칼로리에 어떻게 나타나는지 보여준다"**
GLP-Care만이 구조적으로 가능한 약속. Phase 2 완성 시점(12주차)에 마케팅 전면 부각. 미경 같은 "효과 증명" 유저의 결제 트리거.

**VP3 — "3단계 등록, 알림 한 번, 달력으로 확인"**
Medisafe·CareClinic의 paywall 복잡성과 Shed의 의료 상담 부담 대비 "가벼움"을 약속. 정호·B 같은 디지털 약자의 첫 문턱을 낮춤.

## 2. 수익화 전략

### 2.1 원칙: 무엇을 하지 않을 것인가

경쟁사 리뷰 분석에서 반복적으로 저평가된 패턴을 **의도적으로 회피**한다.

1. **"약 2개 이상 유료" 같은 기본 기능 paywall 금지** — Medisafe가 이 구조로 대대적 욕 먹음. GLP-Care는 약 개수 무제한 무료.
2. **"바코드 스캔 유료" 같은 핵심 동작 paywall 금지** — MyFitnessPal 패턴. 식단 입력 핵심 경로는 무료.
3. **광고 금지 (최소한 Phase 1~2 동안)** — 정호 같은 중장년이 가장 싫어함. 무료 유저도 쾌적하게.
4. **환불 거부·자동갱신 함정 금지** — 앱스토어 표준 결제만 사용.
5. **가격 불투명 금지** — Embla 패턴 회피. 모든 가격은 설정·앱스토어·웹사이트에서 동일하게 노출.

### 2.2 무료 vs 프리미엄 기능 경계

| 약 등록 (무제한) | ✅ | ✅ | ✅ |
| 복약 체크리스트 | ✅ | ✅ | ✅ |
| 푸시 알림 (기본 + 재알림) | ✅ | ✅ | ✅ |
| 달력 뷰 + 주간 요약 | ✅ | ✅ | ✅ |
| 식단/칼로리 기록 (수동 + 공공 DB) | ✅ | ✅ | ✅ |
| 데이터 로컬 저장 (AsyncStorage) | ✅ | ✅ | ✅ |
| **클라우드 동기화 + 기기 교체 복구** | ❌ | ✅ | ✅ |
| **진료용 PDF 월간 리포트** | ❌ | ✅ | ✅ |
| **가족 공유 (읽기전용 링크)** | ❌ | ✅ | ✅ |
| **공유 이미지 자동 생성 (인스타용)** | ❌ | ✅ | ✅ |

| **주사 부위 로테이션 트래커** | ❌ | ❌ | ✅ |

| **증량 D-7 자동 알림** | ❌ | ❌ | ✅ |

핵심 의도:
- 무료 유저도 **앱을 "쓸 수 있다"** — 복약 기록의 모든 핵심 기능. Medisafe 실수 회피.
- 프리미엄은 **"한 단계 위의 활용성"** — PDF, 가족, 상관 인사이트, 백업. 데이터를 "가치로 바꾸는" 레이어.
- 파워팩은 **GLP-1 전용 도메인 깊이** — 일반 복약 유저에게는 불필요하므로 분리 가격.

### 2.3 가격 구조

| 플랜 | 월 | 연(30% 할인) | 평생 |
|---|---:|---:|---:|
| 무료 | 0원 | — | — |
| **프리미엄** | 3,900원 | 29,000원 | — |
| **GLP-1 파워팩** | 7,900원 | 59,000원 | — |
| 평생 이용권(한정 프로모션) | — | — | 99,000원 |

**가격 근거:**
- **3,900원/월:** 한국 건강앱 지불감 기준(팻시크릿 프리미엄 25,300원/연 ≈ 2,108원/월, Medisafe $4.99 ≈ 6,700원/월의 중간대). 인스타 프리미엄·유튜브 뮤직 개인(7,900원)·티빙 베이직(9,500원)보다 저렴해야 첫 결제 진입이 쉬움.
- **29,000원/연:** 월 환산 2,417원. 미경(이미 위고비 월 80만원 지출)에게 "1%도 안 됨" 인식 가능. 정호(결제 경험 적음)에게도 "하루 80원꼴" 네이밍으로 접근.
- **7,900원 파워팩:** 미경·A 같은 GLP-1 파워유저가 "내 약값 80만원 대비 1% 미만이네"로 인식. 프리미엄과 분리한 이유는 일반 복약 유저(정호의 아내 입장)에게 파워팩 기능이 불필요하기 때문. 분리하면 프리미엄 전환율 자체가 높아짐.
- **99,000원 평생:** 초기 100~300명 얼리어답터·인플루언서용. PMF 확인 후 폐쇄. 조기 수익 + 충성 사용자 확보.

**무료 체험:** 프리미엄 첫 **14일 무료**. 파워팩은 **7일 무료**. 길이 차이는 기능 복잡성에 따른 학습 곡선 때문.

**환불:** 앱스토어 표준(Apple 14일·Google 48시간) + 고객센터 요청 시 일반 환불 수용 방침(트위터·앱스토어 리뷰에서 "환불 잘 해준다" 평판 확보가 초기 핵심 마케팅 자산).

### 2.4 페르소나별 결제 전환 시나리오

**페르소나 C (미경) — 결제 의향 높음, 차별화 필요**
- 첫 진입 → 약 등록 → 체크 → 달력 확인 → "이 앱은 무료로 충분해 보임"(무료만 씀)
- 결제 트리거: **약 ↔ 칼로리 상관 대시보드**가 Phase 2에 등장해야 한다. 위고비 맞은 날 전후 칼로리 변화가 그래프로 시각화되는 순간 "이건 MyFitnessPal에서 못 보던 거네" 인지.
- 2순위 트리거: 인스타용 공유 이미지. 미경은 결과를 과시하고 싶다.
- 3순위 트리거: 체중 기록 연동 → "위고비 시작 1개월 -3kg" 자동 생성 카드.
- 권장 플랜: **GLP-1 파워팩 연 59,000원** (월 환산 4,917원).
- 예상 전환율: 14일 체험 → 결제 **12~18%** (GLP-1 특화 기능 완성 시).

**페르소나 D (정호) — 결제 경험 적음, 가족·의료 서사 필요**
- 본인 혼자서는 결제 안 할 확률 80%.
- 결제 트리거 1: **아내가 대신 결제** — 가족 공유 기능 → 아내가 "이거 유용하네, 내가 결제할게" 인식. 이 경로가 현실적 가장 큰 전환 채널.
- 결제 트리거 2: **진료용 PDF** — 내과 진료 전날 "지난 한 달 복약률 95%, 빠뜨린 약 2회" 요약. 의사에게 건네는 순간 "이거 돈 낼 만하네" 인식.
- 권장 플랜: **프리미엄 연 29,000원** (월 2,417원, "하루 80원" 마케팅).
- 예상 전환율: 14일 체험 → 결제 **3~6%** (단, 가족 공유 도입 시 8~12% 기대).

**페르소나 A (30대 직장인, 기존 1차)** — GLP-1 파워팩 타겟. 증량 이력·주사 부위 트래커에 반응. 예상 전환율 15~20%.

**페르소나 B (55세 주부)** — 프리미엄은 관심 적음. 무료 유저로 남을 가능성 높음. 큰 글씨 모드가 프리미엄에 있으면 "자식이 결제해줄" 경로 존재. 예상 전환율 1~3%.

**블렌디드 전환율 목표(GLP-1 유저 + 복약 유저 50:50 가정):** 3개월 내 **3~5%**. 한국 평균 2%(AppsFlyer 한국 헬스앱) 대비 1.5~2.5배. 니치 특화로 달성 가능 범위.

### 2.5 결제 유도 UX 원칙

- **첫 진입 시 결제 유도 금지** — 14일 체험도 자동 시작 아닌 유저 선택 시작.
- **기능 막힘 paywall은 "왜 이 기능이 있는지"를 먼저 설명** (가치 노출 → 전환 유도).
- **다크 패턴 금지** — "지금 결제하지 않으면 데이터 삭제" 등 공포 마케팅 금지.
- **월 결제 해지 방법을 앱 내부에 명시** — 해지 쉬움을 신뢰 자산으로.
- **"어느 순간 결제"**: 진료용 PDF 다운로드 버튼 터치 시, 가족 공유 링크 생성 시, 상관 대시보드 최초 진입 시. 기능의 가치가 유저에게 확인된 직후.
- **14일 체험 종료 D-3 알림** 친절하게: "체험이 곧 끝나요. 만족스럽지 않으면 해지하셔도 돼요." 적대감 제거.

### 2.6 광고 정책

**Phase 1~2(출시~6개월): 광고 0.** 정호·B 같은 중장년의 이탈 요인 1순위. 무료 버전에도 없음.

**Phase 3 이후 재검토.** 만약 도입한다면: 네이티브 광고(배너 아님), 건강 관련 브랜드만, 약물·다이어트 선동 광고 금지. 프리미엄 유저는 광고 제거 자동.

## 8. 3줄 결론

**1. 출시 전 4~6일의 P0 작업이 전부다.** 데이터 영속성·푸시·폰트·터치·어제 배너. 이거 없으면 Phase 1 자체가 시작되지 않는다.

**2. GLP-Care의 진짜 무기는 "약 ↔ 칼로리 상관 인사이트"와 "GLP-1 증량 이력"이다.** 경쟁사 모두 공백이고, 구조적으로 GLP-Care만 가능하다. Phase 2에 반드시 완성.

**3. 수익 모델은 "고가치 프리미엄 + 고투명성 정책"이다.** 무료에서 복약 기록의 모든 기초를 제공하고, 프리미엄은 "데이터를 가치로 바꾸는" 레이어로 설계. 월 3,900원·연 29,000원·파워팩 7,900원이 한국 지불감과 경쟁사 가격 사이의 정확한 위치다. 블렌디드 전환율 3~5%가 3개월 목표.
