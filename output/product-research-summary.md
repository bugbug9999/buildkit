## 1. 한 줄 요약

**GLP-Care는 "한국 GLP-1 자가기록 + 중장년 복약 관리"라는 경쟁 공백 포지션에 있지만, 현재 빌드의 데이터·알림·접근성 기초가 미구현 상태라 네 명의 테스트 페르소나 모두 3개월 내 이탈을 예측했다. 출시 전 6일의 P0 작업으로 참가비를 내고, 이후 "약 ↔ 칼로리 상관 인사이트"와 "GLP-1 증량 이력"이라는 경쟁사 미보유 차별화 무기 2개로 니치 1위를 노리는 것이 전략이다.**

## 3. 주요 발견

### 3.1 페르소나 NPS

| 페르소나 | NPS | 이탈 예측 | 치명타 |
|---|:---:|---|---|
| A 30대 직장인 | (1차 참고) | 중 | 증량 이력·편집 기능 부재 |
| B 55세 주부 | (1차 참고) | 높 | 폰트·터치·10단계 등록 |
| **C 미경 34세** | **3/10** | 1개월 내 삭제 | 데이터 비영구성 + 차별화 부재 → "무료로 충분" |
| **D 정호 46세** | **4/10** | 3개월 85% | 저녁 약 알림 없음 + 데이터 비영구성 |

**공통 1순위:** 데이터가 저장되지 않는다(useState만 사용). 이 한 가지가 모든 페르소나의 신뢰를 무너뜨린다.

### 3.2 경쟁사 맵 핵심

- **Shed / Found / WW GLP-1 / Embla** — 글로벌 GLP-1 앱. 의료팀·처방·배송 포함. 규제 부담 크고 한국 미진출. 월 $20~$299.
- **Medisafe / MyTherapy / CareClinic** — 범용 복약 관리. 칼로리 없음. GLP-1 증량 맥락 없음. Medisafe는 "약 2개 이상 유료"로 욕 먹음.
- **MyFitnessPal / 팻시크릿 / 인아웃** — 칼로리 전용. 약물 데이터 없음. 한국 DB는 팻시크릿·인아웃 강점.
- **삼성 헬스 / PASTA / InBody / Noom** — 종합 헬스앱. GLP-1 특화 없음. PASTA는 혈당 중심.

**경쟁사 매트릭스의 4대 공백 (GLP-Care 기회):**
1. 한국어 × GLP-1 특화 "원앱"
2. 약 + 칼로리 구조적 통합 (단, 현 빌드는 물리적 병존만)
3. 기록만 → 규제 회피 + 빠른 온보딩
4. GLP-1 증량 스케줄의 구조적 이해 (DosageHistory 모델)

### 3.3 시장 타이밍

- 위고비 2024.10 출시 + 2025년 40% 가격 인하로 자가구매 진입장벽 하락
- 2024.12 비대면 처방 중단 → 개인 기록 앱 의존도 ↑
- 오젬픽 2026.2 보험 급여 + 마운자로 2024.7 비만 적응증 → 사용자 풀 확대
- **GLP-Care 타이밍:** 지금이 초기 사용자 확보 창이다. 3~6개월 지체 시 삼성헬스·PASTA 등이 GLP-1 기능 추가할 가능성.

## 4. 핵심 페인포인트 18개 → 우선순위 3단계

| Phase | 작업 | 기간 | 핵심 |
|:---:|---|:---:|---|
| **P0** (출시 차단) | 데이터 영속성, 푸시 알림, 폰트 14px, 터치 44pt, 어제 놓친 배너 | **4~6일** | 참가비 |
| **P1** (출시~6주) | GLP-1 프리셋 템플릿, 연속 등록, 달력 개선, 주간 요약 상세, 온보딩, 다중복약 레이블 | **2~3주** | 기본기 완성 |
| **P2** (6~12주) | **GLP-1 증량 이력, 약↔칼로리 상관 대시보드**, 진료용 PDF, 공유 이미지, 가족 공유, 식단 DB 연동 | **6주** | **차별화 무기** |

P0를 건너뛰고 P2로 직행하면 신뢰가 없어 사용자를 붙잡지 못하고, P2 없이 P0·P1만 하면 경쟁사와 차별화가 없어 결제 전환이 안 된다. **순서가 전략이다.**

## 5. 수익화 핵심

### 5.1 원칙
- 무료 유저도 **앱을 제대로 쓸 수 있게** 한다 — Medisafe의 "약 2개 paywall" 실수 회피
- **광고는 Phase 1~2 동안 0** — 중장년 이탈 1순위
- 투명한 가격, 쉬운 해지

### 5.2 가격

| 플랜 | 월 | 연(30% 할인) | 타겟 |
|---|---:|---:|---|
| 무료 | 0원 | — | 기본 기록, 알림, 달력, 주간 요약 |
| **프리미엄** | 3,900원 | 29,000원 | PDF 리포트, 가족 공유, 상관 대시보드, iCloud 백업 |
| **GLP-1 파워팩** | 7,900원 | 59,000원 | 증량 프로토콜, 주사 부위 트래커, 체중 연동 |
| 평생(프로모션 한정) | — | 99,000원 | 초기 100~300명 |

- 팻시크릿(2,108원/월)보다 약간 비싸지만 약물 기능 추가
- Medisafe(6,700원/월)보다 싸면서 칼로리·GLP-1 추가
- Noom·WW Clinic(2~5만원)의 1/10 가격

### 5.3 결제 전환 목표
- **3개월 블렌디드 구독 전환율 3~5%** (한국 평균 2% 대비 1.5~2.5배)
- 14일 무료 체험 → 결제 25%
- ARPU 월 300원
- 결제 트리거 (페르소나별):
  - **미경(C):** 약↔칼로리 상관 대시보드, 인스타 공유 이미지 → 파워팩 59,000원/연
  - **정호(D):** 가족 공유(아내가 대신 결제), 진료용 PDF → 프리미엄 29,000원/연
  - **A(파워유저):** 증량 이력·주사 부위 → 파워팩

## 6. 3개월 로드맵 요약

```
Week -2 ~ 0  Phase 0 (P0 5종 해결)            → 출시
Week 0 ~ 6   Phase 1 (P1 6종) + 피드백 루프    → PMF 검증
Week 6 ~ 10  Phase 2 (P2 차별화 무기 3개)      → 데이터가 가치로
Week 10 ~ 12 수익화 전환 (프리미엄/파워팩 출시) → 구독 전환율 측정
```

**Go/No-Go Week 6:** D7 리텐션 40%, D30 리텐션 25%, NPS +20, 평균 약 2.5개 등록 중 **3개 달성** → Phase 2 진입.

**Go/No-Go Week 12:** 블렌디드 구독 전환 3% 이상 → 정식 수익화 유지.

## 8. 주요 리스크 3종

1. **P0 작업이 지연되면 모든 후속 단계가 무너진다** — 6일 내 5개 이슈 해결 필수.
2. **경쟁사(삼성 헬스·PASTA)가 GLP-1 기능 추가 시 타이밍 윈도우 닫힘** — 초기 6주 내 500명 획득으로 선점.
3. **규제 해석 변화로 "기록 앱"도 의료기기로 분류될 가능성** — 모든 카피를 "유저가 직접 등록한 기록의 시각화"로 엄격 유지, 약 추천·용량 조언 기능 절대 금지.

## 9. 즉시 실행 권고 (본 리포트 수령 직후)

1. **엔지니어링:** P0 5개 이슈(데이터·푸시·폰트·터치·배너)의 작업자 지정, 4~6일 스프린트 시작
2. **프로덕트:** GLP-1 4개 약물 프리셋 템플릿 데이터 준비, 3단계 온보딩 와이어프레임
3. **마케팅/BD:** 네이버 카페·인스타 GLP-1 커뮤니티 매핑, 내과·비만클리닉 3곳과 전단 협의 착수
4. **리서치:** Week 3에 페르소나 C·D 대상 실제 사용자 5명씩 모집하여 유저 테스트 재검증
5. **CPO:** 본 보고서 기반 출시 일정·가격 정책 의사결정 회의 1회

## 10. 결론

GLP-Care는 **포지션은 정확하지만 현재 구현은 약하다**. 출시 전 6일의 집중 작업과 출시 후 12주의 차별화 스프린트가 제대로 실행되면 한국 GLP-1 니치 1위에 도달 가능한 현실적 경로가 있다. 단, P0를 타협하면 그 경로가 사라진다. 데이터가 저장되지 않는 앱은 앱이 아니다.
